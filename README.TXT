Công nghệ và công cụ sử dụng
Để xây dựng và chạy project quản lý và ứng dụng cơ sở dữ liệu
cho website bán hàng trang sức, nhóm chúng em đã sử dụng một số công nghệ sau:

Visual Paradigm version 17.2: Đây là một công cụ thiết kế và quản lý dự án chuyên
nghiệp hỗ trợ UML, BPMN, và nhiều kỹ thuật mô hình hóa khác. Nhóm sử dụng Visual
Paradigm vẽ các biểu đồ UML để phân tích và thiết kế hệ thống một cách trực quan,
chuyển đổi các yếu tố UML thành mã nguồn để triển khai trong dự án phần mềm.

Apache Netbeans 23 : Netbeans là một môi trường phát triển tích hợp (IDE) được 
sử dụng trong lập trình máy tính. Phiên bản này hỗ trợ nhiều ngôn ngữ lập trình 
và có nhiều plugin mở rộng khả năng của nó. Nhóm sử dụng Netbeans để lập trình 
dựng project bằng ngôn ngữ java.

JDK- 17: Đây là bộ phát triển Java (Java Development Kit) phiên bản 16, cung cấp 
môi trường phát triển cho việc xây dựng web Java. JDK cung cấp các công cụ cho việc 
viết mã nguồn Java, biên dịch mã nguồn thành mã máy thực thi, và tạo các tệp JAR 
(Java Archive) để đóng gói ứng dụng, kiểm tra và gỡ lỗi chương trình.

Tomcat version 9.0: Apache Tomcat là một loại web server HTTP được phát triển bởi 
Apache Software Foundation, nó có khả năng hỗ trợ mạnh cho các ứng dụng Java thay 
vì các website tĩnh khác. Tomcat hỗ trợ các công nghệ như Java Servlet, JSP, WebSocket 
và cung cấp một môi trường ổn định để chạy ứng dụng web.

Maven version 3.9.9 : Apache Maven là một chương trình quản lý dự án cho phép các 
developers có thể quản lý về version, các dependencies ( các thư viện sử dụng trong 
dự án ) , quản lý build, tự động download javadoc & source ... Maven cung cấp các plugin 
để xây dựng ứng dụng, biên dịch mã nguồn, đóng gói thành các tệp JAR hoặc WAR, và triển 
khai lên máy chủ. Nhóm sử dụng Maven để tự động xây dựng và triển khai ứng dụng trên máy
chủ web Apache Tomcat.

MySQL Workbench 8.0 CE : là một công cụ quản lý tích hợp cho MySQL Community Server, giúp
quản lý cơ sở dữ liệu SQL và phát triển các ứng dụng liên quan đến cơ sở dữ liệu. Nhóm sử
dụng MySQL Workbench 8.0 CE tạo và quản lý các bảng, quan hệ, và chỉ mục một cách trực 
quan, thực hiện các truy vấn, cung cấp các công cụ để phân tích và tối ưu hóa hiệu suất 
của cơ sở dữ liệu, giúp trang web hoạt động mượt mà và nhanh chóng.
